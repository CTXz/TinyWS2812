Detailed Description

Blinks one or more WS2812 strips using a RGB array.

Author
Patrick Pedersen
Date
2021-04-09 See

The following example showcases how the Tiny-WS2812 library can be used on AVR platforms supporting the Arduino framework to blink an entire WS2812 LED strip in white. In this rather memory expensive example, we achieve this by simply creating a rgb array the size of the LED strip whose values we then transmit to the strip using the ws2812_tx() function.

For a more memory efficient method, take a look at the blink_loop.c example.

Note
Please ensure that the WS2812_TARGET_PLATFORM_ARDUINO_AVR macro is defined during compilation. This can either be done by specifying -DWS2812_TARGET_PLATFORM_ARDUINO_AVR in the build flags, or by uncommenting the #define WS2812_TARGET_PLATFORM_ARDUINO_AVR directive at the top of this file.
